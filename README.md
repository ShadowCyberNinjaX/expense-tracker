# Student Expense Tracker

A Flask-based web application for managing personal expenses.

## Features
- Add/Delete Expenses
- Monthly Reports
- Budget Tracking
- Savings Estimation
- Category-wise Pie Charts
- SQLite Database

## Tech Stack
- Python
- Flask
- SQLite
- HTML/CSS
- Chart.js

## Future Improvements
- User Authentication
- Excel Export
- Month-wise Analytics
